WAS MACHT DIESE DATEI?

Die Datei TrainingStats.php enthält die Klasse TrainingStats.
Diese Klasse wertet Trainingsdaten eines bestimmten Fahrers statistisch aus.

Sie kann Trainingsdaten nach folgenden Kriterien filtern:
- Fahrer-ID
- Trainingsziel
- Startdatum
- Enddatum

Danach berechnet sie pro Monat verschiedene Kennzahlen:
- Anzahl der Trainings
- Summe der Kilometer
- Durchschnitt
- Minimum
- Maximum
- Median
- 25%-Quantil
- 75%-Quantil
- Standardabweichung

Die Klasse wird nicht direkt im Browser angezeigt.
Sie ist ein Hilfsbaustein für andere Dateien, zum Beispiel für eine Auswertungsseite.

        FROM Training, Fahrer
        WHERE Training.Team = Fahrer.Team

        ORDER BY Training.Datum';

ERKLÄRUNG:

  - Nur der ausgewählte Fahrer wird betrachtet.

- "implode(' AND ', $bedingungen)" verbindet alle Bedingungen mit "AND".
- "ORDER BY Training.Datum" sortiert die Trainings nach Datum.

foreach ($this->monatswerte as $monat => $werte) {

- "durchschnitt" = durchschnittliche Kilometer pro Training.

- "quantil_25" = Wert, unter dem ungefähr 25 Prozent der Werte liegen.

BEISPIEL:
Für Werte [20, 40, 60]:
- Anzahl = 3
- Summe = 120
- Durchschnitt = 40
- Minimum = 20
- Maximum = 60
- Median = 40

ERKLÄRUNG:

public function getMonatswert($monat)
{
    $statistik = $this->getMonatsStatistik();
    return isset($statistik[$monat]) ? $statistik[$monat] : null;
}

ERKLÄRUNG:
- Die Methode berechnet zuerst die Monatsstatistik.
- Danach prüft sie, ob der gewünschte Monat existiert.
- Wenn ja, wird die Statistik dieses Monats zurückgegeben.
- Wenn nein, wird "null" zurückgegeben.
- "null" bedeutet:
  Es gibt keinen Wert.

BEISPIEL:

Das liefert nur die Statistik für April 2026.


==== QUANTIL BERECHNEN ====
public function quantil($werte, $q)
{
    sort($werte, SORT_NUMERIC);
    $position = (count($werte) - 1) * $q;
    $unten = (int) floor($position);
    $oben = (int) ceil($position);

ERKLÄRUNG:
- Ein Quantil beschreibt eine Position innerhalb sortierter Werte.
- "$q" ist der gewünschte Anteil.
- 0.5 bedeutet Median.
- 0.25 bedeutet 25%-Quantil.
- 0.75 bedeutet 75%-Quantil.
- "sort($werte, SORT_NUMERIC)" sortiert die Werte numerisch.
- "$position" berechnet die genaue Position im Array.
- "floor(...)" rundet ab.
- "ceil(...)" rundet auf.


==== QUANTIL: EXAKTE POSITION ====
if ($unten === $oben) {
    return $werte[$unten];
}

ERKLÄRUNG:
- Wenn untere und obere Position gleich sind, liegt das Quantil genau auf einem vorhandenen Wert.
- Dann kann dieser Wert direkt zurückgegeben werden.

BEISPIEL:
Bei [10, 20, 30] und q = 0.5 liegt die Position genau auf 20.


==== QUANTIL: ZWISCHEN ZWEI WERTEN ====
return $werte[$unten] * ($oben - $position) + $werte[$oben] * ($position - $unten);

ERKLÄRUNG:
- Manchmal liegt die berechnete Position zwischen zwei vorhandenen Werten.
- Dann wird zwischen dem unteren und dem oberen Wert interpoliert.
- Interpolieren bedeutet:
  Es wird ein Zwischenwert berechnet.

BEISPIEL:
Bei [10, 20] und q = 0.5 liegt der Median zwischen beiden Werten.
Das Ergebnis ist 15.


==== STANDARDABWEICHUNG BERECHNEN ====
public function standardabweichung($werte)
{
    $durchschnitt = array_sum($werte) / count($werte);
    $summe = 0.0;

ERKLÄRUNG:
- Die Standardabweichung zeigt, wie stark Werte um den Durchschnitt streuen.
- Zuerst wird der Durchschnitt berechnet.
- "$summe" startet bei 0.0.
- Dort werden gleich die quadratischen Abweichungen gesammelt.


==== ABWEICHUNGEN AUFSUMMIEREN ====
foreach ($werte as $wert) {
    $summe += ($wert - $durchschnitt) ** 2;
}

ERKLÄRUNG:
- Die Schleife läuft durch alle Kilometerwerte.
- "$wert - $durchschnitt" berechnet die Abweichung vom Durchschnitt.
- "** 2" quadriert diese Abweichung.
- Quadrieren bedeutet:
  Der Wert wird mit sich selbst multipliziert.
- "$summe +=" addiert den berechneten Wert zur Summe.

WARUM QUADRIEREN?
Dadurch werden negative und positive Abweichungen beide positiv gezählt.


==== STANDARDABWEICHUNG ZURÜCKGEBEN ====
return sqrt($summe / count($werte));

ERKLÄRUNG:
- "$summe / count($werte)" berechnet die durchschnittliche quadratische Abweichung.
- "sqrt(...)" zieht daraus die Quadratwurzel.
- Das Ergebnis ist die Standardabweichung.

BEDEUTUNG:
- Kleine Standardabweichung:
  Die Trainingskilometer sind sehr ähnlich.
- Große Standardabweichung:
  Die Trainingskilometer unterscheiden sich stark.

ZUSAMMENFASSUNG
================================================================================

TrainingStats.php stellt eine Klasse bereit, die Trainingsdaten eines Fahrers auswertet.

Die Klasse:
1. speichert Filter wie Fahrer, Trainingsziel und Zeitraum,
2. lädt passende Trainingsdaten aus der Datenbank,
3. gruppiert Kilometerwerte nach Monat,
4. berechnet statistische Kennzahlen pro Monat.

Wichtige Programmierkonzepte in dieser Datei:
- Klassen und Objekte
- private Eigenschaften
- Konstruktor
- Getter und Setter
- Prepared Statements
- dynamische SQL-Filter
- Arrays
- statistische Berechnungen
